## Overview

A task runner for [my d3 visualizations](https://bl.ocks.org/ch-bu).

## How to run locally

Install node.js, gulp and bower

```
$ npm install
$ bower install
$ npm install -g gulp
```

Serve the application with gulp:

```
$ gulp serve
```

A new tab in your browser should be opened with the following url:

```
http://localhost:3000
```
